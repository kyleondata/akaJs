# akaJs
Alias manager written in JavaScript

### Install
```npm install -g akajs```

then add ```source ~/.aka``` to your ```~/.bash_profile```
and then ```touch ~/.aka```

### Usage

Listing aliases
```aka list```

Adding alias
```aka add <alias> <command>```

Removing alias
```aka remove <alias>```

You can always ask for help
```aka -h```
